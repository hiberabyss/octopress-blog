介绍Gnuplot的一些用法。

<!--more-->

编辑后罪名为`.gp`的文件，用`gnuplot -p your.gp`编译，其中`-p`表示总是显示生成的图像，如果设置生成`eps`格式文件，则不会显示图像，下面是一个示例的`.gp`文件：

```gnuplot
#设置x,y坐标范围
set xrange [0:9]
set yrange [400:900]

#按对数格式显示坐标间隔
set logscale y
#设置表示不同曲线的小列表位置
set key right
#设置x坐标显示
set xtics (1,2,3)

set term postscript font "Times New Roman, 20"

#设置x,y坐标标题文字
set xlabel "X label text"
set ylabel "Y label text"

set terminal postscript eps
set output "mypicture.eps"
#data.txt为数据文件，按行列组织，已空白符号间隔
#using 1:2中2表示第二列数据
plot "data.txt" using 1:2 title "Write" with lp , '' u 1:3 title "Read" w lp

#画柱状图
set style data histogram
set style histogram clustered gap 1
#以颜色进行区分
#set style fill solid 0.4 border
#以斜线进行区分
set style fill pattern 4 border -1
#2:xticlables(1):表示使用第2列数据，同时以第一列为横坐标文字说明
plot "store04-data.txt" u 2:xticlabels(1) title columnheader(2), '' u 3:xticlabels(1) title columnheader(3)
```
